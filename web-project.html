Website Project

Welcome to Pages of Website Projects. Details were provided about the creation of our team website in this section for SITE 1101 course.

Introduction

The website was created as a part of our groupwork in SITE 1101 at ADA University. The main purpose is to show how the HTML, CSS, JAVASCRIPT, and team cooperation using Github are used by our groupworkers.

Technologies Used

  • HTML: To structure website's content.
  • CSS: For desinging the website and making it visually appealing.
  • JavaScript: To add animations and interactivity.
  • Github: For cooperation of team and version control.
  • Visual Studio Codes: As our primary code editor.

Features of the Websites

  • A top navigation bar.
  • Engaging content on events.
  • Interactive elements as animations.
  • Good design that makes sure usability on other various devices.
