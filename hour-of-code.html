Hour of Code Event - Team 40

Team 40 Hour of Code Event

Introduction

Hour of Code is a global event that aims to introduce individuals, especially students, to the basics of computer programming. Before arranging our event, we learned how to organize an Hour of Code from the official website. Following their guidance, we contacted Baku Secondary School No. 329, provided detailed information about the event, and obtained permission to host it. Afterward, we registered our event on the Hour of Code site, selected Minecraft Adventurer as the activity, and planned every detail, including setting the date and creating a schedule.

Event Details

The event was held on November 1st, 2024, at 11:00, at Baku Secondary School No. 329. 28 students from the 6th grade participated, most of whom were beginners. Only 10% had heard about Hour of Code before. The event was divided into three parts:

  1. Presenting ADA University
  2. Hour of Code Activity
  3. Question and Answer Session

Summary

Part 1: Information about ADA University

From 11:00 to 11:15, Farat Babayev, Ismail Aras, and Aysu Suzani gave a presentation about ADA University. We shared detailed information about the campus, faculties, and the education system.

Part 2: Hour of Code

From 11:15 to 11:55,students participated in interactive coding exercises using Code.org. Farat Babayev guided them through puzzles using a projector, while Ismail Aras and Aysu Suzani provided individual assistance. Students were motivated with stickers as rewards for completing levels, which encouraged active participation.

At the end, participants received certificates for their successful completion of the activity.

Part 3: Question and Answer Session

Students asked questions about ADA University, campus life, and IT faculty. Team members shared insights and provided effective answers. Feedback collected indicated that participants enjoyed the event and expressed interest in continuing to learn coding through platforms like Code.org.

Challenges Faced

  • Technical Issues: Some students faced technical difficulties with equipment and internet connectivity, causing minor delays.
  • Diverse Skill Levels: Students had varying levels of experience, making it challenging to meet everyone’s needs in a limited timeframe.
  • Time Constraints: While the one-hour duration was sufficient for an introduction, more time is needed for deeper understanding.

Event Outcomes

Despite the challenges, the event was a great success. Students expressed enthusiasm about continuing their coding journey. Many shared plans to explore more coding puzzles and workshops, highlighting the event’s effectiveness in sparking interest in programming.

Acknowledgments

We thank ADA University for this opportunity and the school administration of Baku Secondary School No. 329 for their support. This event not only introduced coding but also inspired participants to explore programming in greater depth.
